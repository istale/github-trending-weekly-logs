## 觀察

- **AI Agent 基礎設施仍是最大熱點**：本週幾乎一半的 repos 都與 AI Agent 直接相關 — herdr（多 Agent 路由）、orca（Agent 艦隊管理）、strix（AI 滲透測試）、CubeSandbox（Agent 沙盒）、codex-plugin-cc（跨模型協作），說明 Agent 生态正快速往下沉工具層擴展，而非只停留在模型本身。
- **本地化 / 隱私優先成明確趨勢**：meetily（100% 本地處理的會議助手）獲得 22K stars，speech-to-speech（本地語音 Agent）、xbtlin/ai-berkshire 都有這個取向，反映用戶對雲端 AI 隱私成本的警覺。
- **System Prompts 洩漏成為 research 熱點**：system_prompts_leaks 以 56K stars 高居第二，說明研究者、開發者對理解不同 AI 系統行為差異有強烈需求，這類工具短期內會持續受關注。
- **Rust 在基礎設施工具中快速崛起**：本週出現兩個 Rust 專案（meetily、herdr）而非傳統 Python/JS，呼應近年 Rust 在效能敏感工具鏈的滲透。
- **傳統開源專案低調穩定**：immich（自架照片管理）、daisyui（UI 元件庫）、cs249r_book（學術教材）屬於長青專案，不靠 AI 熱點但各自在其領域有穩定關注度，說明 GitHub Trending 並非全部被 AI 佔據。
